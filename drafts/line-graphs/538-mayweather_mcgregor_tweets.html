The Data (and packages)

The mayweather_mcgregor_tweets data come from the fivethirtyeightdata package. You can install it (and the fivethirtyeight package) using the code below.

Extract all_flames

hyped_bored <- hyped_bored %>% 
  mutate(
    flames = str_extract_all(string = emojis_used, pattern = "")
  ) %>% 
  tidyr::unnest_wider(data = .,
    col = flames,
    names_sep = "_") %>%
  tidyr::unite(
    col = "all_flames",
    contains("flames_"),
    sep = " ",
    na.rm = TRUE,
    remove = TRUE

library(zoo)
hyped_bored %>% 
  mutate(
    flame_count = emoji_count(all_flames),
    sleep_count = emoji_count(all_sleeps)
  ) %>% 
  mutate(
    flame_4m_avg = zoo::rollmean(flame_count, k = 4, fill = NA))
